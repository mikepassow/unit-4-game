Crystal Collector!

Welcome to Crystal Collector. In this game you will try and match the random number provided by adding up the different values of the four crystals.
Each crystal will have a random number generated between 1 and 12.
Each matching number equals one win.
Any number that goes over the target number equals one loss.
A score/wins of three completes the game!
Choose each crystal wisely!

Good luck!
